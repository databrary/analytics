Institutions & investigators

Modified

September 3, 2024

This page provides information about Databrary’s partner institutions and the investigators they have authorized.

Institutions

Code 
targets::tar_load(inst_df, store = "../_targets/")

inst_daa_df <- inst_df |>
  dplyr::filter(daa == TRUE)

There are n= 848 total institutions in the Databrary system.

Of these, n= 815 or 96.1% have active Databrary Access Agreements (DAA). Institutions without active DAAs have typically had a single authorized user leave the institution or are institutions that would not typically authorize individual researchers, e.g., government agencies or professional societies.
